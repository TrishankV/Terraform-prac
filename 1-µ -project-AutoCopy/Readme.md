A micro project to make the use of Azure Data Factory to keep track on one container and copy any files added or modified in that container to another container. 
For this i followed a simple process given by GPT , but realised that it had some redundant steps such as inclusion of creation of resources for blobs that caused confusion such that it took me sometime to properly debug the problem. Apart from that always check the spellings for the files that read by terrafomr such as terraform.tfvars.
